# Nutrition App UI Design v0.8 追補 — 週次レポートの過去週ナビ

分析タブの「充足率レポート — 週平均」に週送りを追加する追補
（これまでは今日を含む週のみ）。

## 1. 仕様

- 見出し行の左右に「← 前の週」「次の週 →」（44px タップ）
- 「次の週 →」は今日を含む週で無効（未来週なし・記録タブの日付ナビと同方針）
- 過去週は月〜日の7日をすべて評価対象にする（今週のみ今日まで）。
  API 変更なし: `GET /api/analysis?period=weekly&date=` に
  その週の評価末日（過去週=日曜・今週=今日）を渡す
- 記録が1日もない週は「この週の記録はありません」+ ナビのみ表示
  （欠食日コールアウトは出さない）
- 日次セクション（UL・バー・DG超過）は当日のまま変えない。
  週送りは週次セクション内で完結する
