# decision-20260714-ui-design-adoption

## 決定

デザイン探索「Nutrition UI Explorations」の最終案を MVP の UI 設計として採用する。
詳細は `docs/design/ui_design_v0_1_20260714.md` を正本とする。

## 採用内容

```text
- IA: 3タブ（ホーム・記録・分析）。検索タブは廃止し「記録」内の手段に統合（6a）
- カラー×状態セマンティクス: 1色=1意味 + 冗長符号化（✓・斜線・数値）（5a）
- 不足表示: 達成先出し・上位2件+折りたたみ・主表現「あと◯◯」（5b）
- 超過表示: DG/UL 区別の事実提示。良し悪し判断なし・警告色(赤)不使用（6b）
- 空状態: 初日は「最初の1食」CTA、欠食日は平均から除外し理由明示（6c）
- パレット: Base #FFFDFC / Surface #DDF7F5 / Primary #2F8C7E / Accent #F4C23A(達成限定)
```

## 理由

- UI/UX 評価で挙がった7つの要改善点（色の意味論・密度・冗長符号化・不足フレーミング・
  タップ対象44px・分析→記録導線）をすべて反映済みの最終案であるため
- safe wording policy（診断・助言に見えない事実提示）と表現レベルで整合しているため

## 影響

- `/foods` の独立画面は作らない（`GET /api/foods` は維持）
- ゴールド(accent)の用途を達成バッジ・チップに限定
- デスクトップ案(1c/2a/4c)は MVP 対象外
